## ğŸ¯ Overview

**LexiSwipe** is a comprehensive English vocabulary learning application built with Flutter, designed to help learners master essential English words through an engaging, swipe-based interface. The app features curated word lists from the **Oxford 3000/5000** and **American Oxford** collections, combined with interactive learning modes, pronunciation support, and gamified practice sessions.

Whether you're preparing for language exams, improving your everyday English, or building professional vocabulary, LexiSwipe provides a structured, efficient, and enjoyable learning experience across Android, iOS, and Web platforms.

## ğŸš€ Getting Started

### Prerequisites

Before you begin, ensure you have the following installed:

- **Flutter SDK** (3.0 or higher) - [Installation Guide](https://docs.flutter.dev/get-started/install)
- **Dart SDK** (included with Flutter)
- **Android Studio** with Android SDK (for Android development)
- **Xcode** (for iOS development, macOS only)
- **Git** for version control

### Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/YusufAlper17/LexiSwipe.git
   cd LexiSwipe
   ```

2. **Install dependencies**
   ```bash
   flutter pub get
   ```

3. **Verify Flutter installation**
   ```bash
   flutter doctor
   ```
   Resolve any issues reported by Flutter Doctor before proceeding.

### Running the App

#### Android
```bash
# List available devices
flutter devices

# Launch an emulator
flutter emulators --launch <EMULATOR_ID>

# Run the app
flutter run -d <DEVICE_ID>
```

Common Android device ID format: `emulator-5554`

#### iOS (macOS only)
```bash
# Open Xcode workspace
open ios/Runner.xcworkspace

# Or run directly
flutter run -d <iOS_SIMULATOR_ID>
```

#### Web
```bash
flutter run -d chrome
```

### Building for Production

#### Android APK
```bash
flutter build apk --release
```
Output: `build/app/outputs/flutter-apk/app-release.apk`

#### Android App Bundle (for Play Store)
```bash
flutter build appbundle --release
```
Output: `build/app/outputs/bundle/release/app-release.aab`

#### iOS
```bash
flutter build ios --release
```
Then open Xcode to archive and upload to App Store.

#### Web
```bash
flutter build web --release
```
Output: `build/web/`

## âš™ï¸ Configuration

### Google Mobile Ads

1. **Get AdMob App ID** from [Google AdMob Console](https://admob.google.com/)

2. **Android Configuration**
   
   Edit `android/app/src/main/AndroidManifest.xml`:
   ```xml
   <meta-data
       android:name="com.google.android.gms.ads.APPLICATION_ID"
       android:value="ca-app-pub-XXXXXXXXXXXXXXXX~XXXXXXXXXX"/>
   ```

3. **iOS Configuration**
   
   Edit `ios/Runner/Info.plist`:
   ```xml
   <key>GADApplicationIdentifier</key>
   <string>ca-app-pub-XXXXXXXXXXXXXXXX~XXXXXXXXXX</string>
   ```

4. **Update Ad Unit IDs** in `lib/services/ad_service.dart`

> **Note**: Use test ad IDs during development to avoid policy violations.

### In-App Purchases

1. **Google Play Console**
   - Create products in Play Console
   - Note product IDs (e.g., `premium_monthly`, `premium_yearly`)

2. **App Store Connect**
   - Create in-app purchase products
   - Match product IDs with Android

3. **Update Product IDs** in `lib/services/premium_service.dart`

### Text-to-Speech & Audio

- **Android**: Audio permissions are automatically configured by plugins
- **iOS**: Ensure proper audio session categories in `Info.plist`
- **Offline TTS**: Some TTS voices may require download on first use
